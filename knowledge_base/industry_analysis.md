# 行业分析框架知识库

## 行业分类

### 按周期性分类
1. **周期性行业**: 钢铁、煤炭、有色、航运
2. **防御性行业**: 医药、食品、公用事业
3. **成长性行业**: 科技、新能源、医药创新

### 按产业链分类
- **上游**: 原材料、资源
- **中游**: 制造、加工
- **下游**: 消费、服务

## 行业分析维度

### 1. 市场规模
- 当前市场规模
- 市场增长率
- 市场饱和度

### 2. 竞争格局
- 行业集中度 (CR4/CR10)
- 竞争对手分析
- 进入壁垒

### 3. 政策环境
- 产业政策支持
- 监管要求
- 国际贸易政策

### 4. 技术趋势
- 技术成熟度
- 创新方向
- 技术替代风险

## 重点行业框架

### 科技行业
- 技术路线图
- 研发投入占比
- 专利数量
- 人才储备

### 消费行业
- 品牌价值
- 渠道覆盖
- 客户粘性
- 产品矩阵

### 金融行业
- 资产质量
- 资本充足率
- 不良贷款率
- 盈利能力
